## ✨ **한 줄 소개**
> "오늘 할일 앱 만들어줘"라고 말하면, 10초 만에 작동하는 웹앱이 만들어집니다.

---
```

일반인: "아이디어는 있는데 개발을 몰라요"
    ↓
KIVOSY: "프롬프트로 말하세요"
    ↓
AI: "앱이 완성되었습니다"
    ↓
갤러리: "세상에 공개하세요"
    ↓
다른 사람: "와! 이거 좋네요, 어떻게 만들었어요?"
    ↓
프롬프트 복사 → 자신만의 버전 생성
    ↓
커뮤니티 성장
```

## 🎯 **왜 KIVOSY Make인가?**

### 1. **진짜 무료** (BYOK - Bring Your Own Key)
- 자신의 Gemini/Groq API 키를 사용하면 무제한 생성
- 서버비 부담 제로, 영원히 무료

### 2. **10초면 앱 완성**
```javascript
// 프롬프트 예시
"할일 관리 앱 - localStorage 저장, 추가/삭제/완료 체크"
"환율 계산기 (USD, EUR, KRW, JPY)"
"KTX vs SRT 기차표 가격 비교 앱"
```

### 3. **실시간 채팅 수정**
```
🤖 AI: 앱이 생성되었습니다
👤 User: "버튼 색상을 파란색으로 바꿔줘"
🤖 AI: (즉시 수정된 앱 표시)
```

### 4. **갤러리 + 프롬프트 공유**
- 다른 사람이 만든 앱 구경
- "어떤 프롬프트로 만들었는지" 복사 가능
- 나만의 앱도 갤러리에 자동 등록
